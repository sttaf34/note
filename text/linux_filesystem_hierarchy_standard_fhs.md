# /bin
- コマンド用のディレクトリ

# /dev
- デバイスファイル用のディレクトリ

# /etc
- 設定ファイル用のディレクトリ

# /home
- 各ユーザ毎用のディレクトリ

# /lib
- /bin 等で使われるライブラリ用のディレクトリ

# /mnt
- ファイルシステムの一時マウントポイント

# /usr
- 全ユーザのためのアプリケーション

# /var
- 可変(variable)なファイル用のディレクトリ
