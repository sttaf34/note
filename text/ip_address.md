# ローカルループバックアドレス
- 最上位が 127 がローカルループバックアドレスとして予約されている。
- 基本 127.0.0.1 が使われる。localhost が指しているアドレス。

# クラス
- クラスによってネットワーク部とホスト部の境界線が異なる。
- クラスA ネット.ホスト.ホスト.ホスト
- クラスB ネット.ネット.ホスト.ホスト
- クラスC ネット.ネット.ネット.ホスト
- クラスAは、より多くのホストを持てるということになる。
